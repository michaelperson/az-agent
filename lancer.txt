##lancer sonarqube

docker compose up

### Config
Accède à l'interface : http://localhost:9000 (identifiants par défaut : admin / admin).

Change le mot de passe comme demandé.(Test1234=Test1234 par exemple)

Va dans My Account > Security et génère un Token (conserve-le précieusement).

Crée un projet manuellement pour obtenir sa Project Key.

##Configurer Azure DevOps
Créer un lien entre ton portail Azure DevOps (dans le cloud) et l'instance locale.

Dans le projet Azure DevOps, aller dans Project Settings > Service connections.

Cliquer sur New service connection > SonarQube.

Server URL : Entrer l'URL accessible par ton agent (ex: http://localhost:9000 si l'agent est sur l'hôte).

Token : Coller le token généré à l'étape 2.

Donner un nom (ex: SonarQube_Local).
